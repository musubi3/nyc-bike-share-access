About

New York City has one of the largest bike-share systems in the world, but coverage is not evenly distributed. While Manhattan is saturated with stations, deep pockets of the outer boroughs remain disconnected.

This project investigates "Transit Deserts," dense, car-free neighborhoods that lack both subway access and bike-share infrastructure.

Methodology

We utilized a Site Suitability Analysis to move beyond simple visual inspection. Using Python (GeoPandas & Scikit-Learn), we calculated a "Priority Score" for every census tract in NYC based on three weighted factors:

The Expansion Formula

Score = (Density * 0.40) + (Need * 0.45) + (Gap * 0.15)

  • Transit Need (45%):
     Prioritizes neighborhoods with high car-free households.

  • Population Density (40%):
     Ensures new stations serve the maximum number of residents.

  • Network Gap (15%):
     Targets areas furthest from the existing subway/bike grid.

Data Sources

Dataset Source Description

Subway Routes NYU Spatial Data Repository Geospatial lines for active NYC subway routes (2019). Used to visually prove transit gaps.
Neighborhood Tabulation Areas NYC Open Data GeoJSON boundaries for aggregating census tracts into recognizable neighborhoods.

Key Findings

Our spatial analysis identified five specific neighborhoods, including Spring Creek-Starrett City and Parkchester & Soundview-Bruckner-Bronx River, that are high-density, heavily car-free (>60%), and completely unserved by bike share.

Targeting these areas for the next expansion phase would connect over 172,000 households to the transit network.
